Intruciones para empezar a crear el proyecto
y las aplicaciones de nuestro sitio web

Nota: todo tiene que estar ejecutando desde un entorno virtual.

1. Crear el proyecto 
    $ django-admin startproject ModuloMaster ProjectMaster

2. Crear las aplicaciones de nustro servicios
    $ python manage.py startapp appFrontend
